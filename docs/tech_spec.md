Frontend(The "Look"): We will use Next.js and Tailwind CSS. This is like the paint and the slides on the playground that people see.

Backend(The "Brain): We will use FastAPI and Python. This is the hidden engine that makes the playground move.

Database(The "Toy Box") We will use the Supabase. This is the box where we save all the players' answers so we don't lose them.


The Frontend will send the players' answers to the Backend.

The Backend will then put those answers into the Supabase Toy Box.
